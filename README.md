# Spanish Flashcards

A tiny Next.js + TailwindCSS app to practice Spanish vocab with:

- Leitner-style spaced repetition
- Streak tracking
- Mastery levels
- Multiple decks: Basics, Days, Months

## Features
- Flip cards with smooth animations
- Dark & light mode
- Mobile-friendly
- Export & reset progress
